## Instalaciones

Instalar las dependencias de pythoon
```bash
  pip install -r requirements.txt
```


    
## Ejecutar

Ejecutar el archivo el python

```bash
  python main.py
```

Luego seleccionar clima deseado e inicial y por último elegir las variaciones deseadas (done / otonio).
